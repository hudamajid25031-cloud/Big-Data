# 📊 Student Wellbeing Data Pipeline & Interactive SQL Reporting System

## 📌 عن المشروع (About)
مشروع متميز في هندسة وتحليل البيانات (Data Engineering & Analytics) مخصص لمعالجة بيانات الطلاب. يقوم النظام ببناء أنبوب بيانات متكامل **ETL Pipeline** لدمج وتنظيم الجداول الضخمة المتعلقة بأداء الطلاب، صحتهم النفسية، وعاداتهم اليومية، وتحويلها إلى قاعدة بيانات منظمة من نوع **SQLite** لتسهيل اتخاذ القرارات التعليمية والتربوية.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Python** (لغة البرمجة الأساسية للمشروع)
* **Pandas** (لمعاقبة وتطهير وقراءة ملفات الـ CSV وتحليلها)
* **SQLite3** (لإنشاء قاعدة البيانات العلائقية وإجراء استعلامات الربط المعقدة)
* **ETL Architecture** (مفاهيم استخراج، تحويل، وتخزين البيانات)

## 🚀 ميزات النظام والتقارير المتاحة (System Features)
يحتوي النظام على واجهة تفاعلية تتيح استخراج 3 أنواع من التقارير الذكية عبر SQL Queries:
1. **البحث حسب التخصص (Course Filter):** استخراج قائمة الطلاب بناءً على تخصصهم الدراسي.
2. **مؤشر القلق العالي (High Anxiety Alert):** دمج جداول الطلاب مع جداول الصحة النفسية عبر `JOIN` لفرز الطلاب الذين يمرون بمستويات قلق مرتفعة لمساعدتهم.
3. **تحليل العادات والضغط النفسي (Study Hours vs Stress):** استعلام متقدم يربط 3 جداول معاً (`Double JOIN`) لتحليل العلاقة بين عدد ساعات دراسة الطالب اليومية ومستوى الضغط النفسي لديه.
